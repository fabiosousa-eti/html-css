Principal formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tab B (não semantica).

nesta frase, temos um termo em destaque usando a tab STRONG (semantica)

Itálico / Ênfase

Nesta frase, temos um termo em italico usando a tab I (não semantica).

Nesta frase, temos um termo em ênfase usando a tab EM (semantica).

Texto marcado

Podemos criar também um texto um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno ness parágrafo

Texto deletado

Podemos carcar um texto como excluido para indidar que ele deve ser lido, mas não considerado. usando a tag DEL.

Texto inserido

Podemos criar um texto como inserido para indicar que ele foi adicionado. usando a tag INS.

Existe tambem o sublinhado com a tag U (não semantica)

Texto sobrescrito

Podemos criar um texto tio X20x3 usando a tag SUP.

Texto subscrito

Podemos criar um texto tipo H2O usando a tag SUB.
